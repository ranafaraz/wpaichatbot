== Description ==
A ChatBot widget using ChatGPT APIs.

== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/chatgpt-widget` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Use the Settings->ChatGPT Settings screen to configure the plugin.

== Frequently Asked Questions ==

You can get an API key by signing up at OpenAI.
