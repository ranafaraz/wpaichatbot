# WPAIChatBot

**WPAIChatBot** is a WordPress plugin that integrates with OpenAI to provide an AI-powered chatbot for your website. This plugin allows website visitors to interact with a chatbot that can assist with answering questions, providing information, or any other conversational task powered by the OpenAI GPT model.

## Features
- Seamless integration with WordPress.
- Easy setup and configuration via the WordPress admin panel.
- Customizable chat widget to match your website's branding.
- AI chatbot responses powered by OpenAI API.
- Adjustable OpenAI API parameters (temperature, max tokens, etc.).
- Optional conversation history for maintaining context.
- Multi-language support.
- Mobile responsive chat interface.

## Installation

Follow these steps to install and activate the plugin:

1. **Download the Plugin**: Download the latest version of the plugin from the [GitHub Releases page](https://github.com/ranafaraz/wpaichatbot/releases).
2. **Upload to WordPress**: Go to your WordPress admin dashboard, navigate to `Plugins` > `Add New` > `Upload Plugin`, and upload the downloaded plugin zip file.
3. **Install and Activate**: Click "Install" and then "Activate" to enable the WPAIChatBot on your website.
4. **Configure OpenAI Settings**: After activation, navigate to the `AI Chatbot` settings page under the WordPress dashboard to enter your OpenAI API key and configure the plugin.

## Configuration

### Setting Up OpenAI API Key

1. Sign up or log in to [OpenAI](https://beta.openai.com/) and obtain your API key.
2. In the WordPress admin panel, go to `Settings` > `AI Chatbot` and paste your OpenAI API key into the corresponding field.
3. Adjust additional settings, such as chat behavior, welcome messages, and chatbot personality.

### Customizing the Chatbot Appearance
You can customize the appearance of the chatbot by modifying the following options in the settings panel:
- **Chatbox Colors**: Set the color of the chat window, button, and text.
- **Chat Widget Position**: Choose whether the widget appears in the bottom-right or bottom-left of the screen.
- **Branding**: Upload your logo or choose a custom avatar for the chatbot.

## Usage

Once installed and configured, the WPAIChatBot will automatically appear as a chat widget on your WordPress site. Visitors can click the widget to start interacting with the AI-powered chatbot.
